Egy modern, responsive webalkalmazás fejlesztése React keretrendszerben, amely a The Meal DB API-t használja receptek böngészéséhez és mentéséhez. A projekt célja egy gyors és intuitív felhasználói felület kialakítása volt a konyhai folyamatok segítésére.

Főbb technikai megoldások:
  Állapotkezelés (State Management): A React Context API alkalmazása a kedvenc receptek globális kezelésére és az adatok Local Storage-ban történő perzisztens tárolására.

  Hálózatkezelés: Dinamikus adatlekérdezés és keresés megvalósítása külső REST API használatával (fetch, async/await).

  Komponens-alapú architektúra: Moduláris felépítés kialakítása (Layout, Navbar, RecipeCard), amely elősegíti a kód újrafelhasználhatóságát és karbantarthatóságát。

  Responsive UI: Esztétikus és mobilbarát felület megvalósítása egyedi CSS alapokon.
