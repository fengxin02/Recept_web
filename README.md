## 🇬🇧 English Version

### About The Project
This is a modern, responsive web application built with **React** that allows users to browse, search, and save their favorite recipes. It utilizes the public **TheMealDB API** to provide a fast and intuitive user interface designed to assist with everyday cooking, meal planning, and discovering new dishes.

### Key Features
* **Dynamic Data Fetching:** Seamless integration with a REST API to search for recipes and retrieve detailed instructions, categories, and ingredients using modern asynchronous JavaScript (`fetch`, `async/await`).
* **State Management & Persistence:** Employs the **React Context API** to globally manage the user's favorite recipes. The data is persistently stored in the browser's **Local Storage**, ensuring that your favorite recipes are saved across sessions and page reloads.
* **Component-Based Architecture:** Built with a highly modular structure (including components like `Layout`, `Header`, `RecipeCard`, and `RecipeList`), promoting code reusability, separation of concerns, and easier maintainability.
* **Responsive UI:** Features a custom-designed, aesthetic, and fully mobile-friendly user interface built from scratch with standard CSS.

### Built With
* **React.js** (Hooks, Context API)
* **JavaScript (ES6+)**
* **REST API** (TheMealDB)
* **HTML5 & Custom CSS**

## 🇭🇺 Magyar Verzió

### A Projektről
Ez egy modern, responsive webalkalmazás **React** keretrendszerben fejlesztve, amely a **TheMealDB API**-t használja receptek böngészéséhez, kereséséhez és mentéséhez. A projekt célja egy gyors és intuitív felhasználói felület kialakítása volt a mindennapi konyhai folyamatok, az étkezések megtervezése és az új ételek felfedezésének segítésére.

### Főbb Funkciók
* **Dinamikus adatlekérdezés:** Külső REST API integrációja a receptek kereséséhez és a részletes információk lekéréséhez modern aszinkron JavaScript (`fetch`, `async/await`) használatával.
* **Állapotkezelés (State Management):** A **React Context API** alkalmazása a kedvenc receptek globális kezelésére. A rendszer a böngésző **Local Storage**-át használja az adatok perzisztens tárolására, így a megjelölt kedvencek az oldal újratöltése vagy a böngésző bezárása után is megmaradnak.
* **Komponens-alapú architektúra:** Moduláris felépítés kialakítása (pl. `Layout`, `Header`, `RecipeCard`, `RecipeList` komponensek), amely nagymértékben elősegíti a kód újrafelhasználhatóságát és karbantarthatóságát.
* **Responsive UI:** Esztétikus és teljesen mobilbarát felhasználói felület megvalósítása egyedi CSS alapokon, külső UI könyvtárak mellőzésével.

### Használt Technológiák
* **React.js** (Hooks, Context API)
* **JavaScript (ES6+)**
* **REST API** (TheMealDB)
* **HTML5 & Egyedi CSS**
